# Rodwan Bagdadi - Portfolio Showcase

A modern, responsive portfolio website showcasing my journey as a Data Scientist and Machine Learning Engineer.

## 🌐 Live Demo

**[View Portfolio](https://rodwan-portfolio-showcase.vercel.app)** 

## 🚀 About

This portfolio showcases my projects, experience, and skills in data science, machine learning, and software development. Built with modern web technologies, it features:

- **Responsive Design**: Optimized for all devices and screen sizes
- **Dark/Light Theme**: Toggle between themes for better user experience
- **Dynamic Background**: Flowing left-to-right animated elements creating an immersive experience
- **Interactive Project Modals**: Click any project card to view detailed information, achievements, and technical specifications
- **Professional Experience**: Comprehensive timeline of work experience with interactive elements
- **Skills Visualization**: Technical skills with proficiency indicators and interactive categories
- **Smooth Animations**: Custom CSS keyframes and GPU-accelerated transitions
- **Contact Integration**: Direct email integration and professional social links
- **Performance Optimized**: Fast loading with optimized assets and efficient animations

This portfolio was built to showcase my work as a Mechatronics and AI enthusiast — but more importantly, it's designed to be reusable.

## 📁 Project Structure

```
src/
├── components/         # React components
│   ├── ui/            # Reusable UI components (shadcn/ui)
│   ├── About.tsx      # About section with personal introduction
│   ├── Hero.tsx       # Landing section with profile and key info
│   ├── Projects.tsx   # Interactive project showcase with modals
│   ├── Experience.tsx # Professional timeline and achievements
│   ├── Skills.tsx     # Technical skills with category filtering
│   ├── Contact.tsx    # Contact information and social links
│   ├── BackgroundElements.tsx  # Dynamic flowing animations
│   └── BackToTop.tsx  # Smooth scroll-to-top functionality
├── contexts/          # React contexts (Theme management)
├── hooks/            # Custom React hooks for interactions
├── lib/              # Utility functions and configurations
├── pages/            # Page components and routing
└── utils/            # Helper functions and asset management
```

## 🚀 Deployment

This portfolio is deployed on **Vercel** with the following features:

- **Automatic Deployments**: Connected to GitHub for seamless CI/CD
- **Custom Domain**: Professional domain setup with SSL
- **Performance Optimization**: Optimized builds with Vite and asset compression
- **SEO Optimization**: Meta tags, structured data, and social media previews
- **Global CDN**: Fast loading worldwide with edge caching

### 🔧 **Deploy Your Own**

1. **Fork this repository** to your GitHub account
2. **Connect your Vercel account** to GitHub
3. **Import the project** in Vercel dashboard
4. **Deploy with default settings** - no configuration needed!
5. **Customize the content** by editing the component files
6. **Update the resume PDF** in the `public` folder

### 📝 **Customization Guide**

- **Personal Info**: Edit `src/components/Hero.tsx` and `src/components/About.tsx`
- **Projects**: Update the projects array in `src/components/Projects.tsx`
- **Experience**: Modify work history in `src/components/Experience.tsx`
- **Skills**: Adjust technical skills in `src/components/Skills.tsx`
- **Contact**: Update social links in `src/components/Contact.tsx`
- **Colors**: Customize theme in `src/index.css` and Tailwind config
